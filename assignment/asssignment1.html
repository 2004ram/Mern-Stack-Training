HTML Text Formatting Tags

Tag Description Example
<b> Defines bold text. This is bold text.
<strong> Defines important text. This is strong text.
<i> Defines italic text. This is italic text.
<em> Defines emphasized text. This is emphasized text.
<mark> Defines marked or highlighted text. This is marked text.
<small> Defines smaller text. This is small text.
<del> Defines deleted (strikethrough) text. This is deleted text.
<ins> Defines inserted (underlined) text. This is inserted text.
<sub> Defines subscripted text. This is subscripted text: H2O
<sup> Defines superscripted text. This is superscripted text: E=mc2
<u> Defines underlined text. This is underlined text.
<abbr> Defines an abbreviation or acronym. HTML
<cite> Defines the title of a work. This is a citation.
<code> Defines a piece of computer code. This is code text.
<kbd> Defines keyboard input. Ctrl+C
<samp> Defines sample output from a computer program. This is sample output.
<var> Defines a variable in programming or math. x
